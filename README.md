# crawel-blind-date-website-informaion
一个基于ruby爬取汇总相亲网站信息的程序

## 爬取相亲网站的信息 

爬取的网站：   https://m.7799520.com/api/recommend/wap/search/list/search

相关参数
- startage 筛选年龄的最小值
- endage  筛选年龄的最大值
- startheight 筛选身高的最小值
- endheight 筛选身高的最大值
- salary 工资
- page 分页
